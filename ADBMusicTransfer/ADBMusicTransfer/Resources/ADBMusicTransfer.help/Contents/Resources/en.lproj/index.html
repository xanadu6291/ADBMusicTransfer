ADBMusicTransfer Help

1. Getting ADB and installing ADB

This apprication requires ADB (Android Debug Bridge) to run. Install it with the following procedure:

  1. Download ADB from the link: Android SDK Platform-Tools
  2. Unzip the file, and place adb into the PATH passed directory(Ex: /usr/local/bin)
  3. Check your adb with adb devices in Terminal.app.

USB debug must be enabled on your Android device.

2. UI Parts

  • Select ADB Manually - Can select adb on your Mac manually. (Nomally, the app detects adb aautomatically.)
  • Confirm ADB Connection - Confirm Android device and your Mac connection.
  • Storage Type to transfer Music files: - Can choose strage type to transfer music files on youre Andrroid device.
  • Airtist - Text field to input Airtist name. (Have suggestions)
  • Add files as New Artist - Checkbox to treat airtist as new airtist.
  • Album - Text field to input Album name. (Have suggestions)
  • Transfer Music - Transfer choosed music folder to Android device.
  • Log - Display command execution result/error.

3. Suggestions

When you enter the artist name and album name, the auto-completion candidate appears.

  • Candidates are displayed in real time according to the input characters.
  • If the input is pasted, the suggestion will not be displayed.
  • When there is only one candidate, the input will be automatically confirmed.

4. About music transfer

The adb push command is used for transfer. Copy the music folder of the selected artist album to the device.

If the transfer is successful, "✅ success" will be displayed, and if it fails, "⚠️ error" will be displayed in the output field.
